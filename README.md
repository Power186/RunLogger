# RunLogger
Track your runs in real-time on a map, save and view logged runs on a map

## Built with:
- Xcode
- Swift
- UIKit
- MapKit
- CoreLocation
- Realm
- Realm Database
- ShimmerSwift
- MVC
- Swift Package Manager
